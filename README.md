# Daily Reporter Skill —— 职场汇报智能生成助手

> 把你的碎片化工作记录，自动转化为专业、结构化、成果导向的日报/周报/月报。

## 这是什么？

`daily-reporter` 是一个 AI Agent Skill，它能帮你在 3 分钟内将零碎的工作记录转化为一份专业的汇报文档。

**它不只是润色你的文字——它会重新组织你的信息逻辑。**

## 核心特性

- 🎯 **智能分类**：自动识别核心产出、项目推进、日常运营、问题风险
- 📊 **三种模式**：支持日报、周报、月报，各有专属模板
- 🔄 **语言升级**：将"改了几个 Bug"自动转化为"修复 X 项系统缺陷，提升系统稳定性"
- 📋 **OKR 逻辑**：按目标与关键结果的逻辑重新编排内容
- ✅ **验收清单**：内置质量检查，确保每份汇报都达标

## 快速开始

### 安装

将本项目克隆到你的 Skill 目录：

```bash
git clone https://github.com/freestylefly/daily-reporter.git
```

或手动将 `SKILL.md` 文件复制到你的 Agent Skills 目录中。

### 使用

只需提供你的原始工作记录，然后告诉 AI 你要生成什么类型的汇报：

```
帮我写一份日报，素材如下：
1. 早上开了个会，讨论了新版本的UI，大家觉得按钮太小了
2. 下午联系了3个供应商，A报价200，B报价180，C没回
3. 修了2个上周遗留的Bug
4. 明天打算写完测试报告
```

### 输出效果

```markdown
### 2026-04-07 工作日报

📌 今日核心成果
- **完成新版本 UI 评审**：收集交互体验核心反馈，已列入优化清单
- **供应商成本调研推进**：初步锁定两家意向供应商，对比报价优势明显

📋 项目进展
- **新 UI 设计**：完成评审会议，确定按钮尺寸优化方案
- **供应链管理**：对接 3 家供应商，已获得 2 份报价（180-200 元区间）
- **系统维护**：修复 2 项历史遗留缺陷，提升系统稳定性

📅 明日计划
- 完成并提交核心功能测试报告
```

## 进阶用法

### 指定角色和汇报对象

```
帮我写一份周报，我是产品经理，汇报给技术VP，本周OKR目标是完成V2.0需求评审。
```

### 使用公司自定义模板

```
帮我按以下模板写月报：
[粘贴你公司的模板]
```

### 从聊天记录生成

你可以直接把微信/飞书的聊天记录粘贴过来，Skill 会自动提取有效信息。

## 设计理念

**工具负责从 0 到 80，你负责从 80 到 100。**

- 📌 **痛点即入口**：不从"这个工具是什么"开始，而从"你现在被什么卡住了"开始
- 📋 **模板即答案**：不给原理，给填空题
- ✅ **标准即安全感**：告诉你"做到什么程度算完成"
